# 📌 Projects Included

# 1. 🎮 Number Game
A simple guessing game where the computer generates a random number and the player tries to guess it.

# Features
- Generate a random number between 1 and 100.
- User enters guesses.
- Displays whether the guess is too high or too low.
- Continues until the correct number is guessed.
- Limits the number of attempts.
- Supports multiple rounds.
- Displays score based on attempts and rounds won.

# Concepts Used
- Random class
- Loops
- Conditional statements
- User input using Scanner
